# Dataset

This project was trained using publicly available hand histories generated by the Pluribus poker AI.

The dataset is **not included** in this repository, as it originates from third-party sources.

The data can be obtained from:

- Kevin Wang, *Let's Analyze Pluribus's Hands*
- https://www.kevinwang.us/blog/lets-analyze-pluribuss-hands/

After downloading the files, place them inside the `data/` directory before running the project.
